# Cadastro de Disciplinas

Sistema de gerenciamento escolar desenvolvido em Java com o objetivo de registrar e manter o cadastro de disciplinas, professores e cursos de uma instituição de ensino.

## Requisitos Funcionais

- R1: O sistema deve manter o cadastro de professores. As informações do professor a serem guardadas são: código funcional, nome e data de nascimento.

- R2: O sistema deve manter o cadastro de cursos. As informações do curso a serem guardadas são: código, nome curso e descrição.

- R3: O sistema deve manter o registro de disciplinas. Cada disciplina deve conter: número, nome da disciplina, data de início, data de encerramento, o professor responsável pela
disciplina e o curso ao qual pertence à disciplina.

- R4: Validação: Ao registrar uma disciplina é necessário validar a data de início do curso para que não seja possível informar uma data anterior à data corrente.(método de disciplina).
